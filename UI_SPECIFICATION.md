# 園芸アプリ UI仕様書

## 概要

本ドキュメントは、園芸管理PWAアプリケーションのUI設計を定義する。

### 設計方針

- **モバイルファースト**: スマートフォンでの操作を最優先に設計
- **水やり最優先**: 最も頻繁に使う水やり記録を最短手順で実行可能に
- **シンプル**: 必要最小限の画面遷移で目的を達成
- **PWA対応**: ホーム画面追加、オフライン動作

### 対応デバイス

| デバイス | 対応状況 | 備考 |
|----------|----------|------|
| スマートフォン | ◎ 優先 | ボトムタブナビゲーション |
| タブレット | ○ | スマホUIで動作 |
| PC | △ 将来対応 | 別UIを将来検討 |

## ナビゲーション構造

### 全体構成

```
┌─────────────────────────────────┐
│ 園芸アプリ              ≡      │  ← ヘッダー（右上: ハンバーガーメニュー）
├─────────────────────────────────┤
│                                 │
│                                 │
│         メインコンテンツ         │
│                                 │
│                                 │
├──────────┬──────────┬───────────┤
│   🌱    │   💧    │    📍    │  ← ボトムタブ
│  植物   │  水やり  │   場所   │
└──────────┴──────────┴───────────┘
```

### ボトムタブ

| タブ | アイコン | 遷移先 | 説明 |
|------|----------|--------|------|
| 植物 | 🌱 | 植物一覧 | 管理中の植物を表示 |
| 水やり | 💧 | 水やり画面 | 水やり記録（メイン機能） |
| 場所 | 📍 | 場所一覧 | 場所の管理 |

### ハンバーガーメニュー

| 項目 | 遷移先 | 説明 |
|------|--------|------|
| データエクスポート | エクスポート画面 | JSONファイルをダウンロード |
| データインポート | インポート画面 | JSONファイルを読み込み |
| （将来）Google Drive連携 | 連携設定画面 | 同期設定 |
| このアプリについて | About画面 | バージョン情報等 |

## 画面一覧

### 一覧系画面

| 画面名 | 種別 | URL | 説明 |
|--------|------|-----|------|
| 植物一覧 | タブ画面 | `/plants` | 管理中の植物をリスト表示 |
| 場所一覧 | タブ画面 | `/locations` | 登録済みの場所をリスト表示 |
| 水やり | タブ画面 | `/watering` | 水やり記録のメイン画面 |

### 詳細系画面

| 画面名 | 種別 | URL | 説明 |
|--------|------|-----|------|
| 植物詳細 | ページ | `/plants/:id` | 植物情報と各種記録 |
| 場所詳細 | ページ | `/locations/:id` | 場所情報と環境データ |

### モーダル

| 画面名 | 表示元 | 説明 |
|--------|--------|------|
| 植物追加 | 植物一覧 | 新規植物の登録 |
| 植物編集 | 植物詳細 | 植物情報の編集 |
| 場所追加 | 場所一覧 | 新規場所の登録 |
| 場所編集 | 場所詳細 | 場所情報の編集 |
| 水やり確認 | 水やり画面 | 対象確認と記録実行 |
| 水やり記録追加 | 植物詳細 | 個別の水やり記録 |
| 開花記録追加 | 植物詳細 | 新規開花サイクルの開始 |
| 開花記録編集 | 植物詳細 | 開花サイクルの更新 |
| イベント追加 | 植物詳細 | 成長イベントの記録 |
| 環境データ追加 | 場所詳細 | 気温・湿度の記録 |

### 設定系画面

| 画面名 | 種別 | URL | 説明 |
|--------|------|-----|------|
| エクスポート | ページ | `/settings/export` | データのエクスポート |
| インポート | ページ | `/settings/import` | データのインポート |
| About | ページ | `/about` | アプリ情報 |
